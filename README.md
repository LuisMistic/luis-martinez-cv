# Luis Martinez | CV Landing

Landing curriculum estatica para presentar el perfil full stack de Luis Sebastian Martinez Morinigo, con foco en logistica, sistemas web, proyectos reales y diferencial audiovisual.

## Estructura

- `index.html`: contenido principal.
- `styles.css`: diseno responsive.
- `script.js`: menu mobile y respaldo de imagen.
- `assets/foto-luis.png`: foto principal.
- `assets/LuisCV-original.pdf`: CV PDF original enlazado desde la landing.

## Deploy

El proyecto esta preparado para Firebase Hosting.

```powershell
firebase deploy --only hosting
```

## Actualizar foto

Reemplazar `assets/foto-luis.png` por una nueva imagen con el mismo nombre.
